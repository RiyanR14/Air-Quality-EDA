# Air Quality Data Analysis (EDA & Visualization)

## Project Overview
This project performs Exploratory Data Analysis (EDA) and visualization on air quality data to understand pollution trends across different cities in India.

## Dataset
- Source: Air Quality city_day dataset
- Features include AQI, PM2.5, PM10, NO2, SO2, CO, and more.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analysis
- Missing value analysis and data cleaning
- AQI distribution and pollution levels
- City-wise and time-based AQI trends
- Correlation between pollutants
- Data visualization using charts and plots
